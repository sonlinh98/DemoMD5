# DemoMD5

Các bước chạy:
1. Run spring boot application: mvn spring-boot:run
2. Run Client: 
                cd client
                 yarn serve
                 
Note: Bạn phải cài maven và node js.

Cài maven:
https://mkyong.com/maven/how-to-install-maven-in-windows/

Cài node js:
